## Project Purpose
This project uses Python to analyze the risk and return performance of five major U.S. technology stocks.
Data is obtained from the WRDS CRSP database for academic analysis.

## Data Source
- Database: WRDS (Wharton Research Data Services)
- Dataset: CRSP Daily Stock Data
- Period: 2023–2024
- Stocks: AAPL, MSFT, GOOGL, AMZN, TSLA

## Analysis Contents
- Data loading and preprocessing
- Descriptive statistics of returns
- Annualized return and volatility calculation
- Sharpe ratio analysis
- Data visualization (price trends, risk‑return scatter, key metrics)

## Python Libraries
- pandas
- numpy
- matplotlib
